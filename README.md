## Problem
A school administrator needs a simple, console-based tool to manage student records and course enrollments. The current process is manual making it difficult to track which students are enrolled and in which courses.

This program should solve that problem by providing a menu-driven interface that allows the administrator to:

- Add a new student (with ID and name)
- Remove a student by ID
- Search for a student by ID
- Display all currently enrolled students
- Add a course to a specific student's record
- View all courses a student is enrolled in

The application is built using core Java concepts including Object-Oriented Programming (OOP), a `HashMap` for efficient student lookup, and `Scanner` for console input.
